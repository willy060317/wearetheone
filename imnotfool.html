📋 프로젝트 주제

Team1.AI는 Flask를 활용해 심리테스트 설문조사 폼을 제작하는 프로젝트입니다. 🌌 사용자가 설문에 참여하면, 입력 데이터를 분석해 맞춤형 결과를 제공하는 웹 애플리케이션을 개발했어요. Flask로 RESTful API를 설계하고, SQLAlchemy로 SQLite 데이터베이스를 관리하며, Bootstrap과 JavaScript로 직관적인 UI를 구현했습니다. 🚀

예를 들어, "술 선호도" 설문을 통해 사용자의 주류 선호도와 소비 패턴을 분석, 개인화된 추천을 제공합니다. 이 과정에서 백엔드는 데이터를 효율적으로 처리하고, 프론트엔드는 사용자 친화적인 경험을 제공해요. 🧠

🎯 프로젝트 목표

Git 협업

Git을 활용해 팀원들과 소스 코드를 관리하며, 브랜치 전략과 Pull Request로 협업 프로세스를 익혔습니다. 🗂️

Flask 환경 설정

Flask 프로젝트 초기화, 가상 환경 설정, pip으로 종속성 관리를 복습하며 안정적인 환경을 구축했어요. ⚙️

Route와 함수

@app.route 데코레이터로 URL과 함수를 연결, RESTful API 엔드포인트를 설계했습니다. 🌐

SQLAlchemy

SQLAlchemy ORM으로 SQLite DB에 설문 데이터(질문, 선택지, 답변)를 저장/조회/분석했어요. 🗄️

📽️ PPT 제작 과정: 임우진의 도전

이 프로젝트를 소개하기 위해, 임우진은 전통적인 PPT 제작을 시작했어요. 하지만 곧 한계에 부딪혔죠. 😴 슬라이드 기반의 PPT는 정적이고 단방향이라, 프로젝트의 역동성과 기술적 깊이를 제대로 전달하기 어려웠습니다. Flask의 빠른 API, SQLAlchemy의 유연한 DB 관리, Git 협업의 생동감을 PPT로만 보여주는 건 부족하다고 느꼈어요.

임우진은 단순히 정보를 나열하는 대신, 사용자가 프로젝트를 **체험**할 수 있는 방법을 고민했어요. 그래서 선택한 게 **AI 챗봇 UI**였습니다. 챗봇은 PPT의 틀을 깨고, 사용자가 키워드를 입력하면 실시간으로 반응하며 프로젝트의 핵심을 동적으로 보여줄 수 있죠. 예를 들어, "술 선호도"를 입력하면 설문 안내와 버튼이, "어려운"을 입력하면 기술 스택 이미지가 나타나요. 🌌

이 챗봇은 단순한 대화형 인터페이스를 넘어, 프로젝트의 기술적 성취와 팀의 비전을 몰입감 있게 전달합니다. PPT 슬라이드 하나로는 불가능했던 **사용자 중심의 인터랙티브 경험**을 제공하며, Team1.AI의 혁신성을 강조해요. 🚀

🤖 임우진의 비전: 챗봇 혁신

임우진의 비전은 단순한 프로젝트 소개를 넘어, 사용자가 Team1.AI의 기술과 열정을 **직접 느끼게** 하는 거였어요. 🌠 전통적 PPT는 슬라이드 전환과 텍스트로 제한되지만, AI 챗봇은 실시간으로 사용자와 소통하며 프로젝트의 모든 면을 생동감 있게 보여줍니다.

예를 들어, 사용자가 "고마워"를 입력하면 스타워즈 스타일의 `crawl.html`이 새 창으로 열리고, "질문"을 입력하면 API 코드 이미지가 나타나죠. 이 챗봇은 텍스트, 이미지, 버튼, 링크를 동적으로 제공하며, 사용자를 프로젝트의 중심으로 끌어들입니다. 👾 임우진은 챗봇을 통해 **은하계 수준의 몰입감**을 구현하고, 동적 PPT의 한계를 완전히 깨부쉈어요.

이 혁신은 Team1.AI의 핵심 철학을 반영합니다: 기술은 단순히 도구가 아니라, 사람과 세상을 연결하는 다리예요. 챗봇은 사용자가 프로젝트를 탐험하며 배우고, 즐기고, 영감을 받는 경험을 제공합니다. 🚀
